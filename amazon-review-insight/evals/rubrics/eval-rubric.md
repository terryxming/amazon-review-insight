# Eval Rubric

| 维度 | 分值 |
| --- | ---: |
| 证据忠实度 | 35 |
| unknown 使用 | 15 |
| Review 编码完整度 | 12 |
| 关键结论分布质量 | 12 |
| 主题粒度和归因质量 | 13 |
| 业务动作可执行性 | 15 |
| 报告和 Excel 契约一致性 | 5 |

通过线：

- 单个 eval 得分达到 80。
- 8 个 eval 平均分达到 85。
- 证据忠实度单项不得低于 32。
- 编造 evidence、统计分母错误、非 unknown 字段无 evidence、关键结论缺少 distribution、Excel 编码层缺少必需 sheet 时直接失败。
